# Contact Manager 
This application allows users to find Piranha Technologies on the web


## Getting Started
You will need to run npm install in terminal for all dependencies

### Prerequisites

Node.JS,
React

## Deployment
to run server type npm start after downloading dependencies

## Built With


JavaScript,
Express,
Node,
React,
Googlemaps API
